# Cálculo Inc, proyecto cálculo 2 Jala U
---

### Ejecutar en Processing

* Instalar Processing última versión: https://processing.org/
* Ir al proyecto en calculo-inc\PlagueInc
* Ejecutar: PlagueInc.pde (se debería abrir Processing)
* En el IDE seleccionar la opción de ejecutar arriba a la izquierda
* Para cerrar presionar 'Detener' que está al lado del anterior

Alternativa:

* Abrir Processing
* Ir a la parte de 'Archivo' -> 'Abrir'
* Ir al proyecto
* Ir a PlagueInc
* Seleccionar PlagueInc.pde
* Abrir
* En el IDE seleccionar la opción de ejecutar arriba a la izquierda
* Para cerrar presionar 'Detener' que está al lado del anterior
